Sitio Web de la tienda de vinilos ficticia "El Vinilo de Vasiliy" en Tacna-Perú, creado con HTML, CSS, bootstrap, JavaScript, AJAX y Json, se implementa manejo del DOM para la carga inicial de las Cards de productos y para la incorporación de filtros por categoría solicitado por el cliente, teniendo como ventaja que se carga primero cada página y luego se cargan las cards de los productos, de esta manera garantizamos una mejor experiencia de usuario.
